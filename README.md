# Shorts
- [Live demo](https://shorts-mu.vercel.app/)
- Hosted with vercel
- Made with create-react-app

## Setup

```
# clone the repo
  gh repo clone alwinantony29/shorts
```
```
# go to shorts directory
  cd shorts
```
```
# install node modules
  npm i 
```

## Run

```
# start server
  npm start
```
